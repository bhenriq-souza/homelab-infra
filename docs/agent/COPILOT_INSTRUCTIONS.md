# Copilot Instructions

## Regras
- trabalhar em pequenas mudanças
- respeitar o escopo do repositório
- não misturar infraestrutura com aplicação
- manter documentação coerente com a implementação
- preferir soluções simples na fase inicial

## Em toda resposta de implementação
Informe:
- arquivos alterados
- resumo da mudança
- como validar
- próximos passos sugeridos
