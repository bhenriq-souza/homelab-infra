# DNS and TLS

## Objetivo
Definir estratégia mínima de nomes e certificados para:
- acesso administrativo
- Grafana
- Argo CD
- serviços do app
- endpoints cloud

## Estratégia inicial
- nomes simples e previsíveis
- priorizar acesso privado nas fases iniciais
- TLS onde houver exposição de interface administrativa
