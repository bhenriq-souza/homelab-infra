# Hybrid Connectivity

## Objetivo
Estabelecer conectividade entre o ambiente on-prem e a VPC da GCP.

## Fases
### Fase inicial
Conectividade simples para administração e experimentação controlada.

### Fase evolutiva
Conectividade mais próxima de um cenário site-to-site.

## Requisitos
- rotas sem conflito
- conectividade previsível
- documentação de portas e fluxos
