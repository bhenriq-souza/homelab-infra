# Documentation

Este diretório centraliza a documentação estrutural do laboratório.

## Seções
- `architecture/`: visão de solução, roadmap e convenções
- `adr/`: decisões arquiteturais registradas
- `network/`: topologia, IPs, DNS, exposição e conectividade híbrida
- `security/`: baseline de segurança e estratégia de segredos
- `operations/`: operação inicial, checklists e runbooks
- `backlog/`: plano de trabalho por fase
- `agent/`: instruções para uso com agentes de IA

## Princípios
- manter documentos curtos e objetivos
- evitar duplicação de conteúdo
- atualizar a documentação sempre que houver mudança estrutural
