# Phase 02 - Network

## Objetivo
Definir e documentar a base de rede do laboratório.

## Macrotarefas
- mapear a LAN atual
- definir IP fixo do mini PC
- definir ranges do cluster local
- definir VPC e subnet cloud
- definir estratégia de conectividade híbrida
- criar matriz de exposição

## Critérios de aceite
- IP plan documentado
- estratégia híbrida documentada
- serviços administrativos com política de acesso definida
